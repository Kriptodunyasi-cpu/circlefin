# Circlefin Community Library 🛠️

This repository is a community-driven resource hub created for developers building on the **Circle** ecosystem and **Arc Network**. Our mission is to contribute to this next-generation financial infrastructure by providing open-source tools, integration guides, and practical examples.

> [!IMPORTANT]
> **Disclaimer:** This repository is not affiliated with, maintained, authorized, or endorsed by the official Circle (circlefin) entity. It is an independent, community-led initiative designed to support the developer ecosystem.

---

## 🚀 Available Modules

Current projects and documentation hosted within this library:

### ⚡ [Arc Network Developer Guide](./arc-network/README.md)
A comprehensive technical guide on smart contract deployment, programmable wallets, and micro-payment systems on Arc Network.
* **Status:** Under Active Development (Arc Testnet)
* **Scope:** Wallet integrations, x402 protocol architecture, and transaction management.

## 🤝 How to Contribute

This library is built on the vision of "Building Together." You can contribute in several ways:

1.  **Code Samples:** Share utility scripts or boilerplate projects using Circle SDK or Arc Network.
2.  **Documentation:** Improve existing guides, fix typos, or add new use-case scenarios.
3.  **New Modules:** Add documentation regarding bridging operations between Circle and other networks (Base, Ethereum, etc.).

## 🛠️ Tech Stack & Tools

The projects within this library generally utilize:
* **Infrastructure:** Circle Programmable Wallets, Smart Contract Infrastructure.
* **Networks:** Arc Network, Base, Scroll.
* **Development:** Solidity, TypeScript, Hardhat, Ethers.js.
